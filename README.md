# Mallorca Map

노션에 임베드할 수 있는 마요르카 53개 municipio 방문 지도입니다.

## GitHub에 올리기
1. 저장소에서 **Add file → Upload files**
2. 이 폴더의 `index.html`을 업로드하고 **Commit changes**
3. **Settings → Pages**
4. `Build and deployment`에서 **Deploy from a branch**
5. Branch를 `main`, 폴더를 `/(root)`로 선택하고 **Save**
6. 1~3분 뒤 아래 주소로 접속:
   `https://MAT1D-debug.github.io/mallorca-map/`

## 노션에 넣기
노션에서 `/embed`를 입력한 뒤 위 주소를 붙여넣으세요.

방문 상태는 해당 브라우저의 localStorage에 저장됩니다.
